## Lo que una auditoría SEO completa debe incluir

![Lo que cubre una auditoría SEO completa con 6 categorías](/images/blog/seo-audit-checklist-categories.webp)

La mayoría se enfoca en SEO técnico. Una auditoría completa cubre 6 áreas.

### SEO técnico

- [ ] Errores de rastreo e indexación
- [ ] Enlaces rotos (internos y externos)
- [ ] Cadenas y bucles de redirección
- [ ] Validez del sitemap XML
- [ ] Configuración robots.txt
- [ ] Implementación HTTPS
- [ ] Precisión de tags canónicas

### SEO on-page

- [ ] Title tags (únicos, bajo 60 caracteres)
- [ ] [Meta descripciones](/es/blog/write-meta-descriptions/) (145-155 caracteres)
- [ ] Jerarquía de headers (H1 > H2 > H3)
- [ ] Texto alt en imágenes
- [ ] [Optimización de keywords](/es/blog/keyword-optimization-guide/)

### Calidad de contenido

- [ ] Contenido fino (bajo 300 palabras)
- [ ] Duplicados
- [ ] [Decay de contenido](/es/blog/content-decay-fix/)
- [ ] [Señales E-E-A-T](/es/blog/eeat-for-blogs/)

### Velocidad

- [ ] Core Web Vitals (LCP, INP, CLS)
- [ ] Rendimiento móvil
- [ ] Optimización de imágenes
- [ ] Tiempo de respuesta del servidor

### Perfil de backlinks

- [ ] Total de dominios de referencia
- [ ] Backlinks tóxicos
- [ ] Backlinks perdidos
- [ ] Distribución de anchor text

### SEO local (negocios locales)

- [ ] Precisión de [Google Business Profile](/es/blog/google-business-profile-optimization-complete-guide/)
- [ ] Consistencia NAP
- [ ] [Targeting de keywords locales](/es/blog/local-seo-guide/)
- [ ] Volumen y velocidad de reseñas

## FAQ

**¿Cuál es la mejor herramienta gratis?**

Google Search Console porque da datos directos de Google. Para crawl técnico profundo, combínalo con Screaming Frog free (500 URLs) y Ahrefs Webmaster Tools.

**¿Cada cuánto correr una auditoría?**

Auditoría técnica completa trimestral. On-page mensual para páginas de tráfico alto. Monitoreo automatizado para problemas críticos en tiempo real.

**¿Cuánto cuesta una auditoría profesional?**

DIY con free cuesta cero. Pagas de $18/mes (Sitebulb Lite) a $500+/mes (Lumar). Agencia para auditoría única: $1.000-5.000 medianas, $10.000+ enterprise. Stacc incluye optimización SEO en curso por $99/mes con 30 artículos.

**¿Qué debe incluir una auditoría?**

6 áreas: SEO técnico, SEO on-page, calidad de contenido, velocidad, backlinks y SEO local si aplica.

**¿Puedo hacer una auditoría yo mismo?**

Sí. Empieza con GSC para indexación y CWV. Corre Screaming Frog free para crawl de 500 páginas. Sigue nuestra [guía de auditoría SEO](/es/blog/how-to-do-seo-audit/).

**¿Semrush o Ahrefs es mejor para auditorías?**

Semrush tiene auditoría más profunda (140+ checks, AI Search Health, mejor reporting). Ahrefs tiene la base de backlinks más fuerte. Para auditoría pura, Semrush gana. Para combinar con backlinks, Ahrefs.
