## What a Complete SEO Audit Should Include

![What a complete SEO audit covers showing 6 audit categories](/images/blog/seo-audit-checklist-categories.webp)

Most audit tools focus on technical SEO. A complete audit covers 6 areas.

### Technical SEO

- [ ] Crawl errors and indexing issues
- [ ] Broken links (internal and external)
- [ ] Redirect chains and loops
- [ ] XML sitemap validity
- [ ] `robots.txt` configuration
- [ ] HTTPS implementation
- [ ] Canonical tag accuracy

### On-Page SEO

- [ ] Title tags (unique, under 60 characters, keyword-optimized)
- [ ] [Meta descriptions](/blog/write-meta-descriptions) (145-155 characters, compelling)
- [ ] Header hierarchy (H1 > H2 > H3, no skipping)
- [ ] Image alt text on all images
- [ ] [Keyword optimization](/blog/keyword-optimization-guide) (placement and density)

### Content Quality

- [ ] Thin content pages (under 300 words)
- [ ] Duplicate content issues
- [ ] [Content decay](/blog/content-decay-fix) (pages losing rankings)
- [ ] [E-E-A-T signals](/blog/eeat-for-blogs) (author bios, sources, expertise)

### Page Speed

- [ ] Core Web Vitals (LCP, INP, CLS)
- [ ] Mobile performance scores
- [ ] Image optimization
- [ ] Server response time

### Backlink Profile

- [ ] Total referring domains
- [ ] Toxic or spammy backlinks
- [ ] Lost backlinks
- [ ] Anchor text distribution

### Local SEO (for local businesses)

- [ ] [Google Business Profile](/blog/google-business-profile-optimization-complete-guide) accuracy
- [ ] NAP consistency across citations
- [ ] [Local keyword](/blog/local-seo-guide) targeting
- [ ] Review volume and velocity

## FAQ

**What is the best free SEO audit tool?**

Google Search Console is the best free SEO audit tool because it provides data directly from Google. For deeper technical crawling, combine it with Screaming Frog's free version (500 URLs) and Ahrefs Webmaster Tools. Together, these 3 free tools cover approximately 80% of what paid tools offer.

**How often should you run an SEO audit?**

Run a full technical audit quarterly. Run on-page content audits monthly for high-traffic pages. Set up automated monitoring (most paid tools offer this) to catch critical issues like broken pages or indexing errors in real time between full audits.

**How much does a professional SEO audit cost?**

DIY with free tools costs nothing. Paid tools range from $18/mo (Sitebulb Lite) to $500+/mo (Lumar). Hiring an SEO agency for a one-time audit costs $1,000-5,000 for mid-sized sites and $10,000+ for enterprise sites. Stacc includes ongoing SEO optimization for $99/mo with 30 published articles.

**What should an SEO audit include?**

A complete audit covers 6 areas: technical SEO (crawl errors, redirects, sitemaps), on-page SEO (titles, meta descriptions, headers), content quality (thin content, duplicates, E-E-A-T), page speed (Core Web Vitals), backlink profile (toxic links, anchor text), and local SEO (GBP, citations, reviews) if applicable.

**Can I do an SEO audit myself?**

Yes. Start with Google Search Console to identify indexing issues and Core Web Vitals problems. Run Screaming Frog's free version to crawl up to 500 pages for technical issues. Use Google PageSpeed Insights for individual page performance. Follow our [SEO audit guide](/blog/how-to-do-seo-audit) for a step-by-step walkthrough.

**Is Semrush or Ahrefs better for SEO audits?**

Semrush offers deeper technical audit features (140+ checks, AI Search Health, better reporting). Ahrefs offers the strongest backlink database and always-on crawling. For audit-focused work, Semrush edges ahead. For combined audit + backlink analysis, Ahrefs is stronger. Both are excellent. Read our [Semrush review](/reviews/semrush) and [Ahrefs review](/reviews/ahrefs) for detailed comparisons.
