## O que é um link interno?

Um link interno é um hyperlink que aponta de uma página do seu site para outra página no mesmo domínio. Ajudando usuários a navegar seu site e ajudando mecanismos de busca a entender sua estrutura de conteúdo.

Todo site usa links internos. Seu menu de navegação, links de rodapé, referências cruzadas em posts de blog e widgets de barra lateral todos contam. Mas os links internos que mais importam para [SEO](/glossary/seo) são os contextuais. Links colocados naturalmente dentro do conteúdo do corpo que conectam páginas relacionadas. Esses são os links que o Google pesa mais fortemente ao determinar como suas páginas se relacionam e quais são mais importantes.

Um estudo da Semrush de 300.000 sites encontrou que páginas com mais links internos apontando para elas tendem a ranquear mais alto. As páginas de melhor ranking no conjunto de dados deles tinham 3,5x mais links internos que páginas ranqueando na página 2. Linkagem interna é um dos poucos fatores de ranking que você controla completamente. Sem outreach, sem espera, sem orçamento necessário.

## Por que links internos importam?

Links internos são o tecido conectivo do seu site. Sem eles, o Google trata suas páginas como ilhas isoladas.

- **Distribuem [link equity](/glossary/link-equity)**. Quando um site externo linka para sua homepage, links internos passam essa autoridade para páginas mais profundas. Sem eles, suas páginas internas não recebem nada desse poder de ranking.
- **Ajudam o Google a descobrir e [indexar](/glossary/indexing) páginas**. O Googlebot segue links internos para encontrar novas páginas. Páginas órfãs (sem links internos apontando para elas) são significativamente menos prováveis de serem indexadas.
- **Estabelecem [autoridade tópica](/glossary/topical-authority)**. Interligar conteúdo relacionado mostra ao Google que seu site cobre um tópico de forma abrangente, não superficialmente. Assim funcionam [silos de conteúdo](/glossary/content-silo) e [clusters de tópicos](/glossary/topic-clustering).
- **Reduzem taxa de rejeição**. Usuários que encontram conteúdo linkado relevante na sua página têm mais probabilidade de ficar e navegar. Links internos mantêm pessoas no seu site mais tempo, o que é um sinal positivo de engajamento.

Se você está publicando conteúdo mas não interligando, está construindo uma biblioteca sem sistema de catálogo. Os livros estão lá, mas ninguém. Inclusive o Google. Pode encontrar o que precisa.

## Como links internos funcionam

Linkagem interna afeta tanto como usuários navegam quanto como mecanismos de busca avaliam seu site.

### Passando autoridade (fluxo de PageRank)

O algoritmo [PageRank](/glossary/pagerank) original do Google foi construído sobre links. Links internos distribuem essa autoridade pelo seu site. Uma página com 10 links internos apontando para ela recebe mais "votos" que uma com 1. Por isso sua homepage. Que tipicamente tem mais links. Geralmente é sua página de mais alta autoridade.

### Sinais de anchor text

O texto clicável de um link interno ([anchor text](/glossary/anchor-text)) diz ao Google do que se trata a página de destino. Linkar para seu "guia de email marketing" com anchor text "email marketing" dá ao Google um sinal de relevância. Usar "clique aqui" como anchor text desperdiça esse sinal inteiramente.

### Criação de caminhos de rastreamento

O Googlebot descobre páginas seguindo links. Sua [estrutura de linkagem interna](/glossary/site-architecture) cria caminhos pelo seu site. Páginas mais profundas que 3 cliques da homepage são mais difíceis para o Google encontrar e rastrear. Links internos estratégicos criam atalhos que trazem páginas importantes mais perto da superfície.

### Distribuição de link equity

Cada página tem uma quantidade finita de autoridade para passar pelos seus links de saída. Uma página com 5 links internos de saída divide sua autoridade mais uniformemente que uma com 50. Isso não significa que menos links é sempre melhor. Mas significa que links de páginas de alta autoridade no seu site carregam mais peso.

## Tipos de links internos

Diferentes tipos de links internos servem a propósitos diferentes:

- **Links de navegação**. Itens de menu, links de cabeçalho, links de rodapé. Aparecem em cada página e dizem ao Google (e usuários) quais páginas são mais importantes em todo o site.
- **Links contextuais/editoriais**. Links colocados dentro do conteúdo do corpo, apontando para páginas relacionadas. Carregam o maior peso SEO porque são tematicamente relevantes e cercados de contexto.
- **Links de breadcrumb** Navegação de [breadcrumb](/glossary/breadcrumbs) mostrando a posição da página na hierarquia do seu site (Início > Blog > Categoria > Post). Ajuda o Google a entender a estrutura do site.
- **Links de barra lateral e widget**. Posts relacionados, artigos populares e links de categoria em barras laterais. Úteis para usuários mas carregam menos peso SEO que links no conteúdo.
- **Links de rodapé**. Links em todo o site no rodapé. Bons para páginas utilitárias importantes (Política de Privacidade, Contato, Sobre). Muitos links de rodapé diluem seu valor.

Links contextuais dentro do seu conteúdo são os que mais movem a agulha do SEO. Todo o resto é infraestrutura de apoio.

## Links internos vs. links externos

Esses servem a propósitos fundamentalmente diferentes.

| | Links Internos | [Links Externos](/glossary/external-link) |
|---|---|---|
| **Destino** | Outra página no seu domínio | Uma página em domínio diferente |
| **Propósito SEO** | Distribui autoridade, estabelece estrutura | Cita fontes, constrói confiança |
| **Você controla** | Completamente. Adicionar, editar, remover a qualquer momento | Apenas saída; entrada (backlinks) depende de outros |
| **Boas práticas** | Use [anchor text](/glossary/anchor-text) descritivo, link contextualmente | Linke para fontes autoritativas, use com moderação |
| **Risco** | Excesso de links dilui valor | Linkar para sites de spam prejudica credibilidade |
| **Exemplo** | `/blog/guia-email-marketing` → `/glossary/email-marketing` | Sua página → `moz.com/blog/guide` |

Você precisa de ambos. Links internos estruturam seu site. Links externos mostram ao Google que seu conteúdo é bem pesquisado e conectado à web mais ampla.

## Boas práticas com links internos

- **Linke de páginas de alta autoridade para alvos importantes**. Sua homepage e posts de blog de melhor desempenho têm mais autoridade. Links internos dessas páginas para conteúdo novo ou de baixo desempenho passam poder de ranking real.
- **Use anchor text descritivo** "Aprenda mais sobre [pesquisa de palavras-chave](/glossary/keyword-research)" é muito mais útil que "clique aqui." O [anchor text](/glossary/anchor-text) deve descrever o tópico da página de destino.
- **Mantenha páginas importantes a 3 cliques da homepage**. É a regra clássica de [arquitetura de site](/glossary/site-architecture). Páginas enterradas a 5+ cliques são mais difíceis para o Google rastrear e ranquear.
- **Adicione links internos retroativamente a conteúdo novo**. Quando publicar um novo artigo, volte a 3-5 páginas existentes relacionadas e adicione links ao novo. A maioria esquece esse passo. O processo de publicação da theStacc lida com a interligação automaticamente. Cada novo artigo linka para e de conteúdo existente relevante no seu site.
- **Audite páginas órfãs trimestralmente**. Use um crawler como Screaming Frog ou Sitebulb para encontrar páginas com zero links internos. Cada página importante deve ter pelo menos 2-3 links internos apontando para ela.

## Perguntas Frequentes

### Quantos links internos uma página deve ter?

Não há número mágico. O Google disse que não há limite rígido. Um post de blog de 2.000 palavras pode incluir naturalmente 8-15 links internos. A chave é relevância. Cada link deve fazer sentido em contexto. Não adicione links apenas para atingir um número.

### Links internos ajudam SEO?

Absolutamente. Links internos são um fator de ranking confirmado. Distribuem autoridade, ajudam o Google a [rastrear](/glossary/crawling) e [indexar](/glossary/indexing) suas páginas, fornecem contexto tópico através de anchor text, e criam a estrutura do site que o Google usa para entender sua hierarquia de conteúdo.

### Devo usar nofollow em links internos?

Quase nunca. [Nofollow](/glossary/nofollow-link) em links internos previne autoridade de fluir para essas páginas. A única exceção são links para páginas de login ou outras páginas que você não quer que o Google priorize. Para páginas de conteúdo, sempre use links internos dofollow padrão.

### O que é uma página órfã?

Uma página órfã é uma página no seu site com zero links internos apontando para ela. O Google pode nunca descobri-la, e mesmo se descobrir, a página não recebe autoridade do seu outro conteúdo. Conserte páginas órfãs adicionando links contextuais de conteúdo relacionado.
