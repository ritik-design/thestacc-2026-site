## Wirklich kostenlose Tools (nie Bezahlung erforderlich)

### 1. Google Keyword Planner

**Am besten für:** Suchvolumen validieren und PPC-Keyword-Ideen finden.

Google Keyword Planner ist das einzige kostenlose Tool, das Suchvolumendaten direkt von Google liefert. Erfordert ein Google Ads-Konto (kostenlos zu erstellen, keine Ausgaben nötig). Die Daten sind PPC-orientiert, aber das Suchvolumen gilt gleichermaßen für SEO.

**Was Sie kostenlos erhalten:**

- Suchvolumenbereiche für jedes Keyword (z. B. 1K-10K monatliche Suchen)
- Keyword-Ideen basierend auf Seed-Keywords oder URLs
- Wettbewerbsniveau (niedrig, mittel, hoch)
- Durchschnittliche CPC-Daten
- Prognosen für Werbeplanung

**Einschränkungen:**

- Volumen wird in Bereichen gezeigt, nicht in exakten Zahlen (außer Sie schalten Anzeigen)
- Wettbewerbsmetrik spiegelt PPC-Wettbewerb wider, nicht SEO-Schwierigkeit
- Oberfläche für Werbetreibende, nicht SEO-Praktiker
- Keine Backlink-Daten, keine SERP-Analyse

**Profi-Tipp:** Google Keyword Planner zeigt exakte Volumen, wenn Sie eine Kampagne einrichten (auch ohne Ausgaben). Erstellen Sie eine Kampagne, fügen Sie Keywords hinzu und prüfen Sie den Prognose-Bericht. Das exakte Volumen erscheint in der Prognose. Pausieren Sie die Kampagne, bevor sie läuft. Sie erhalten exakte Daten zum Nulltarif.

### 2. Google Search Console

**Am besten für:** Keyword-Möglichkeiten auf Ihrer bestehenden Site finden.

Google Search Console ist das am meisten unterschätzte kostenlose Keyword-Recherche-Tool. Es zeigt Ihnen die exakten Suchanfragen, für die Ihre Site bereits rankt, zusammen mit Impressionen, Klicks, CTR und durchschnittlicher Position. Kein anderes Tool liefert diese Daten – nicht einmal Ahrefs oder Semrush.

**Was Sie kostenlos erhalten:**

- Jede Suchanfrage, für die Ihre Site in der Google-Suche erscheint (mit exakten Impressionszahlen)
- Klickrate pro Keyword
- Durchschnittliche Ranking-Position pro Keyword
- Daten auf Seitenebene (welche Seite rankt für welche Keywords)
- 16 Monate historische Daten

**Einschränkungen:**

- Funktioniert nur für Sites, die Sie besitzen (verifiziert über DNS, Meta-Tag oder HTML-Upload)
- Keine Wettbewerber-Keyword-Daten
- Daten verzögern sich um 2-3 Tage
- Sie können keine Keywords erforschen, für die Sie noch nicht ranken

**Wie Sie es zur Keyword-Recherche nutzen:** Gehen Sie zu Performance > Suchergebnisse. Filtern Sie nach „Durchschnittliche Position" zwischen 8 und 20. Das sind Keywords, für die Sie bereits auf Seite 1-2 ranken, aber Verbesserungspotenzial haben. Optimieren Sie diese Seiten, um in die Top 5 zu kommen.

**Erweiterte Technik:** Exportieren Sie Ihre GSC-Daten und sortieren Sie nach Impressionen (höchste zuerst). Keywords mit hohen Impressionen aber niedrigen Klicks sind Ihre größten Möglichkeiten. Schreiben Sie das Title-Tag um, aktualisieren Sie die Meta-Beschreibung und fügen Sie mehr Tiefe hinzu.

Prüfen Sie auch den „Seiten"-Tab. Wenn mehrere Seiten für dasselbe Keyword erscheinen, haben Sie [Keyword-Kannibalisierung](/de/blog/what-is-keyword-cannibalization/).

Für eine vollständige Anleitung lesen Sie unseren [Google Search Console Leitfaden](/de/blog/google-search-console-guide/).

### 3. Google Trends

**Am besten für:** Saisonale Muster verstehen und Themenpopularität vergleichen.

Google Trends zeigt, wie sich das Suchinteresse über die Zeit ändert. Es liefert keine absoluten Volumenzahlen. Stattdessen zeigt es relatives Interesse auf einer 0-100-Skala.

**Was Sie kostenlos erhalten:**

- Interesse über die Zeit (5 Jahre Daten oder Echtzeit für die letzten 7 Tage)
- Geografische Aufschlüsselung
- Verwandte Suchanfragen und aufstrebende Themen
- Side-by-Side-Vergleich von bis zu 5 Begriffen
- YouTube-Suchdaten (umschalten auf „YouTube-Suche")

**Einschränkungen:**

- Keine absoluten Suchvolumenzahlen
- Relative Skala macht Low-Volume-Keywords unsichtbar
- Keine Keyword-Schwierigkeit oder CPC-Daten
- Keine Wettbewerbsanalyse auf Seitenebene

**Wie Sie es zur Keyword-Recherche nutzen:** Vergleichen Sie Ihr Ziel-Keyword mit Alternativen, um zu sehen, welches mehr Nachfrage hat. Prüfen Sie Saisonalität, um Ihren Content zu timen. Nutzen Sie „Aufsteigende" verwandte Suchanfragen, um aufkommende Themen zu finden.

### 4. Keyword Surfer (Chrome-Erweiterung)

**Am besten für:** Keyword-Volumen prüfen, ohne Google zu verlassen.

Keyword Surfer ist eine kostenlose Chrome-Erweiterung, die geschätztes Suchvolumen, CPC und verwandte Keywords direkt in den Google-Suchergebnissen anzeigt.

**Was Sie kostenlos erhalten:**

- Geschätztes monatliches Suchvolumen für Ihre Suchanfrage
- CPC-Daten
- Verwandte Keywords mit Volumen (in einer Sidebar angezeigt)
- Wortzahl und geschätzter Traffic für jede rankende Seite
- Content-Editor für die Artikelplanung

**Einschränkungen:**

- Nur Chrome (funktioniert nicht in Firefox, Safari oder Edge)
- Volumenschätzungen sind weniger genau als Google Keyword Planner für Nischenbegriffe
- Kein Keyword-Schwierigkeitswert
- Keine Backlink- oder Wettbewerbsanalyse

**Was Sie kostenlos erhalten:**

**Einschränkungen:**

- Auf 100 Ergebnisse begrenzt
- Keine Suchvolumendaten in der kostenlosen Version

## Der kostenlose Keyword-Recherche-Workflow

![Kostenloser Keyword-Recherche-Workflow mit 4 Tools, die kostenpflichtige ersetzen](/images/blog/free-keyword-research-workflow.webp)

Kein einzelnes kostenloses Tool ersetzt ein $140/Monat-Semrush-Abo. Aber 4 kostenlose Tools kombiniert decken ungefähr 80 % dessen ab, was kostenpflichtige Tools bieten.

### Schritt 1: Möglichkeiten finden (Google Search Console)

Beginnen Sie mit Keywords, für die Sie bereits ranken. Filtern Sie nach Position 8-20. Das sind Keywords, bei denen eine kleine Content-Verbesserung Sie in die Top 5 schieben könnte.

### Schritt 2: Ideen generieren (AnswerThePublic + Soovle)

Geben Sie Ihr Thema in AnswerThePublic für fragen-basierte Keywords ein. Verwenden Sie Soovle für plattformübergreifendes Brainstorming.

### Schritt 3: Volumen validieren (Google Keyword Planner + Keyword Surfer)

Bringen Sie Ihre Keyword-Liste zum Google Keyword Planner für Volumenbereiche. Verwenden Sie Keyword Surfer zur Spot-Prüfung.

### Schritt 4: Schwierigkeit bewerten (Ahrefs Free Keyword Generator)

Lassen Sie Ihre Shortlist durch das kostenlose Ahrefs-Tool laufen. Fokussieren Sie sich auf Keywords mit KD unter 20.

### Schritt 5: Auf Content abbilden ([Keyword-Optimierung](/de/blog/keyword-optimization-guide/))

Weisen Sie 1 primäres Keyword und 3-5 sekundäre Keywords jedem geplanten Blog-Beitrag zu.

Dieser Workflow dauert 30-45 Minuten pro Batch von 10-15 Keywords. Er kostet nichts.
